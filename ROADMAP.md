# Merzato Roadmap

## 0.2 — Spatial paintings

- Full two-dimensional Direction Pointer and Codel Chooser traversal.
- Connected-colour-block analysis instead of ordered `data-order` rectangles.
- White-space sliding and black-block collision semantics.
- Raster PNG input in addition to SVG.

## 0.3 — Musical authoring

- Multi-track MIDI selection and explicit melody-track metadata.
- MIDI export from artwork and assembly.
- Rhythmic operands and chords without sacrificing playable scores.
- A score visualizer linked to generated instructions.

## 0.4 — Web application toolkit

- More DOM elements and attributes.
- State, forms, timers, local storage, and component composition.
- Capability-based host permissions for network and browser APIs.
- Sandboxed embeddable playground.

## 0.5 — Artist tooling

- Visual editor with palette validation.
- Live colour-transition and interval debugger.
- Instruction tracing across canvas, score, and VM state.
- Gallery format for sharing executable artworks.

## 1.0 — Stable language profile

- Versioned language specification and conformance suite.
- Stable bytecode or serialized intermediate representation.
- Reproducible package releases and browser distribution.
- Compatibility policy for VM, art, MIDI, and MerzScript profiles.
