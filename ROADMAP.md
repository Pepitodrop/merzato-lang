# Merzato Roadmap

## 1.0 — Stable core — complete

- Stable Assembly, ordered SVG, MIDI interval, MerzScript, CLI, and JavaScript profiles.
- Input validation, structured errors, resource limits, browser capability policy, TypeScript declarations, CI, CodeQL, and automated releases.

## 1.1 — Artist tooling

- Source maps linking VM instructions to canvas blocks and MIDI events.
- Instruction tracing across artwork, score, registers, stack, and heap.
- A visual palette and transition validator.
- More examples and an executable-art gallery manifest.

## 1.2 — Musical authoring

- Richer multi-track selection and explicit melody-track metadata.
- MIDI export from artwork and assembly.
- Rhythmic operands and chord profiles that remain musically playable.
- Score visualization linked to generated instructions.

## 2.0 candidate — Spatial Piet profile

- Full two-dimensional Direction Pointer and Codel Chooser traversal.
- Connected-colour-block analysis, white-space sliding, and black-block collision.
- Raster PNG input in addition to SVG.
- An explicit profile selector so 1.x ordered-SVG programs remain reproducible.
