# Changelog

All notable changes to Merzato are documented here. The project follows semantic versioning while the language remains experimental.

## [Unreleased]

## [0.1.0] - 2026-07-13

### Added

- Merzato Assembly parser and stack/register virtual machine.
- Arbitrary-precision integer arithmetic and integer-addressed heap.
- Labels, branches, calls, functions, and deterministic step limiting.
- Piet-style SVG colour-transition compiler.
- Velato-inspired MIDI interval operand mapping.
- MerzScript browser ABI for DOM, event, prompt, logging, styling, and fetch operations.
- Command-line runner, assembler output, source validation, examples, and browser playground.
- Automated tests for assembly, heap/control flow, artwork, MIDI, browser DOM integration, and rapid event serialization.
