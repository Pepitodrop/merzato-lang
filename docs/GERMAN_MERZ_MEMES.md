# German Merz meme example

`examples/merz-memes.mza` uses recognisable German Friedrich Merz memes and recurring political topics as program data:

- `Was ist Bubatz?`
- `Jetzt darf auch mal Rambo Zambo im Adenauer-Haus sein.`
- `Bierdeckel-Steuer`
- `Brandmauer`
- `Mehr arbeiten`

Run it with:

```bash
node src/cli.js run examples/merz-memes.mza
```

In the browser playground, select **Load German Merz memes** and then **Run assembly**.

The example is political satire and is not affiliated with or endorsed by Friedrich Merz, the CDU, or any broadcaster. It deliberately uses light, broadly recognisable meme material rather than demeaning or disputed statements about groups of people.
